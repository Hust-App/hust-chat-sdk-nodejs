#### Funções
Funções diversas usadas para auxiliar no desenvolvimento em geral.
* `zeroEsquerda` - Adiciona zeros a esquerda de um número.  
* `Timeout` - Retorna um Promise que finalizará depois do tempo informado como parâmetro.
